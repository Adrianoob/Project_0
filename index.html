Cashier.java

Attributes

ArrayList item -> the array of items in the shopping cart

Methods
Return Type Method and Description
voidaddItem(String barcode)
Take barcode and add corresponding item to shopping cart.
voiddeleteItem(String barcode)
Delete an item from shopping cart using barcode. Do nothing if barcode doesn’t exist.
doublesum()
Calculate the total price of items in shopping cart.
Stringprint()
Return the string displaying information about current shopping state (including items, price and tax).
doubletax()
Generate the tax of items in shopping cart automatically.
